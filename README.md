# Diego Bastidas - Portafolio Académico y Profesional

Un portafolio moderno y responsivo que destaca la trayectoria profesional, investigación académica e innovación tecnológica de Diego Javier Bastidas Logroño, Magíster e Investigador en Inteligencia Artificial.

## 🌟 Características

- **Diseño Responsivo**: Optimizado para dispositivos móviles, tablets y escritorio
- **Interfaz Moderna**: Efectos glassmorphism, animaciones suaves y gradientes elegantes
- **Contenido Multilenguaje**: Completamente traducido al español
- **Performance Optimizado**: Arquitectura separada de CSS y JavaScript
- **Navegación Intuitiva**: Menú fijo y barra móvil inferior para fácil acceso
- **Secciones Completas**:
  - Perfil y resumen profesional
  - Impacto de investigación y logros
  - Especialidades académicas
  - Publicaciones destacadas
  - Trayectoria profesional detallada
  - Conferencias y presentaciones
  - Testimonios profesionales

## 📁 Estructura del Proyecto

```
djbastidas.github.io/
├── index.html              # Archivo HTML principal
├── styles/
│   └── styles.css          # Estilos personalizados (CSS)
├── scripts/
│   └── script.js           # Lógica de interactividad (JavaScript)
└── README.md               # Este archivo
```

## 🛠 Tecnologías Utilizadas

- **HTML5**: Estructura semántica moderna
- **Tailwind CSS**: Framework de utilidades CSS para diseño rápido
- **JavaScript Vanilla**: Sin dependencias, interactividad pura
- **Lucide Icons**: Iconografía moderna y escalable
- **Google Fonts**: Familias tipográficas optimizadas (Inter, Plus Jakarta Sans, Playfair Display)

## 🚀 Comenzar

### Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a internet (para cargar CDNs de Tailwind, Lucide Icons y fuentes)

### Instalación Local

```bash
# Clonar el repositorio
git clone https://github.com/djbastidas/djbastidas.github.io.git

# Navegar al directorio
cd djbastidas.github.io

# Abrir con un servidor local (recomendado)
# Opción 1: Python
python -m http.server 8000

# Opción 2: Node.js (http-server)
npx http-server

# Luego acceder a: http://localhost:8000
```

## ⚙️ Personalización

### Cambiar Colores
Editar variables en `index.html` (línea 25-32):
```javascript
colors: {
    primary: '#0f172a',      // Azul oscuro
    secondary: '#4f46e5',    // Índigo
    accent: '#10b981',       // Verde esmeralda
    background: '#f8fafc',   // Gris muy claro
}
```

### Modificar Contenido
- Editar textos directamente en `index.html`
- Actualizar enlaces de redes sociales en el footer
- Cambiar imágenes de perfil y publicaciones

### Ajustar Estilos CSS
Editar `/styles/styles.css` para personalizar:
- Animaciones
- Efectos glassmorphism
- Transiciones
- Responsive breakpoints

### Mejorar Interactividad
Modificar `/scripts/script.js` para:
- Eventos de menú móvil
- Animaciones de scroll
- Comportamiento de navegación

## 📱 Responsive Design

- **Mobile First**: Diseñado primero para móviles
- **Breakpoints**:
  - Mobile: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px
- **Barra móvil inferior**: Navegación rápida en dispositivos pequeños

## 🎨 Elementos de Diseño

### Glassmorphism
Efectos de vidrio esmerilado con:
- `backdrop-filter: blur(12px)`
- Transparencia controlada
- Bordes sutiles

### Animaciones
- **Fade-in**: Aparición suave de elementos
- **Hover-lift**: Efecto de elevación al pasar el cursor
- **Scroll**: Animaciones al hacer scroll
- **Transiciones**: Cambios visuales suaves

### Efectos Visuales
- Gradientes lineales y radiales
- Sombras profundas
- Bordes redondeados elegantes
- Grid de fondo sutil
